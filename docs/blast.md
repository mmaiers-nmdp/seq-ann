# Creating blastn files

> [!NOTE]
> Make sure blastn is properly installed before running!

1)  Download the allele list and the `_gen` and `_nuc` fasta files for
    each locus
2)  Create the blast files from the downloaded IMGT/HLA FASTA files with
    your local BLAST tooling.
3)  Add the new blast files to the seqann/data/blast directory and check
    them in.
